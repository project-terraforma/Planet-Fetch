# Project OKRs

## Objective 1
**Deliver Context File Version 1 with a clearly defined statistical scope.**

### Key Results
- Finalize a structured template including:
  - Context rules
  - A glossary
  - Summary sections
- Include **12 evaluation questions** designed to test the usefulness of the context file.
- Add a section summarizing the **high-level questions the context file is intended to answer.**

---

## Objective 2
**Demonstrate measurable reliability of the context file.**

### Key Results
- Create a **scoring rubric** to evaluate:
  - Accuracy
  - Grounding in dataset statistics
- Test responses using **representative evaluation questions.**
- Document **failure cases and limitations** discovered during testing.

---

## Objective 3
**Strengthen and automate the data workflow.**

### Key Results
- After finalizing **Context File v1**, explore **SQL-based rollups** to generate aggregated metrics programmatically.
- Integrate **preprocessing and normalization steps** into the workflow.
- Document the **complete data transformation pipeline** from raw metrics data to final context file.
